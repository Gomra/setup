# setup

My personal setup for [Elementary OS](https://elementary.io/).

## Install

**1.** Get this project somehow and go to its directory.

```sh
git clone git@github.com:iagodahlem/setup.git ~/setup
cd ~/setup
```

**2.** Run setup. (see [setup](setup.sh))

```sh
./setup.sh
```
